# CodeAlpha Object Detection & Tracking

## Project Description

Object Detection & Tracking is a Computer Vision project developed using Python, OpenCV, YOLOv8, and ByteTrack. The system detects and tracks multiple objects such as people, cars, buses, trucks, motorcycles, and traffic lights in images and videos. Each detected object is assigned a unique ID for tracking across video frames.

## Features

- Object Detection in Images
- Object Detection in Videos
- Multi-Object Tracking
- Real-Time Processing
- Unique Object Tracking IDs
- YOLOv8-based Detection
- ByteTrack Tracking Algorithm

## Technologies Used

- Python
- OpenCV
- YOLOv8
- NumPy
- ByteTrack
